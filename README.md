# PROYECTO MODULOS Y PAQUETES EN PYTHON 

### 多Que aprendimos?

- Crear modulos y paquetes en python
- Importar modulos y paquetes en python
- Usar modulos y paquetes en python

### 多Que problema resuelve cada archivo?
- *__init__.py* : Es el archivo que se encarga de inicializar el paquete
- *cursos.py* : Es el archivo que se encarga de crear los cursos
- *utils.py* : Es el archivo que se encarga de crear las funciones

### 多Que hace setup.py?
- Es el archivo que contiene la informacion del paquete, como su nombre, version, autor, etc. y que dependencias necesita para instalarse con pip.install. Sin esto python identifica que es una carpeta, pero no sabe como distribuirla.

### 多Para que sirve versionar un paquete?
- Sirve para saber que version maneja cada ususario y si se hace un cambio que rompe el codigo viejo, se pueda cambiar el numero de version para que los usuarios esten al tanto de los cambios.
