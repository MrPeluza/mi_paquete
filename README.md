# mi_paquete
¿Qué se aprendió?
Aprendí a crear un paquete con módulos e importarlos a otros archivos python, de esta forma aplicamos la reutilización del código y no tenemos la necesidad de volver a typear funciones, atributos y clases.
También se aprendió a crear una descripción y como subirlo mediante un archivo setup.py a pypi, de esta forma más personas podrían utilizar nuestros códigos de manera sencilla.
